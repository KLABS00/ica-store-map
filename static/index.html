Data Sources

OpenStreetMap (Overpass API)

Crowd-sourced geographic data. We query the Overpass API for all nodes, ways, and relations in Sweden tagged with brand~ICA or name~^ICA. Returns ~1,400+ entries with coordinates, but address data is often incomplete since it depends on volunteer contributors. Anyone can improve the data at openstreetmap.org.

ICA.se (Official)

Official store data extracted from ICA's public store finder page at ica.se/butiker. The page embeds all ~1,287 stores as structured JSON in window.__INITIAL_DATA__, including store name, type (profile), full street address, postal code, city, and precise coordinates. Updated whenever ICA updates their website.

Company Data (Bolagsverket)

The Company tab shows registered company data from Bolagsverket (the Swedish Companies Registration Office) via their public API. ICA stores are franchises — each store is typically owned by an independent entrepreneur operating under the ICA brand. The Company tab shows the parent corporate entities (ICA Gruppen AB, ICA Sverige AB). You can also look up any Swedish company by org number.

Limitations

Both sources are cached server-side for 1 hour. OSM data may include closed stores, duplicate entries, or non-grocery ICA entities (e.g. ICA Banken ATMs). ICA data is a point-in-time snapshot and may lag behind real-world openings/closings. Individual store franchise owners can't be matched automatically since the Bolagsverket API requires an org number for lookup — ICA's store data doesn't include franchise org numbers.
